    My first coding project

    I wanted to start my coding journey with a fresh perspective. Using my background in engineering, and previous experience with Python and Matlab, I started out by reading books on how to code, watching various youtube videos across different topics, and googling every question I had under the sun. Overall, I was getting a grasp on what it means to study Computer Science.

    For my first coding project, I thought what better project to have than one that can present all of my upcoming projects. I focused on starting with a static website that can showcase my ability to code, particularly with scripting languages such as HTML and CSS. As i got more familiar with these languages, I started to implement the Bootstrap framework to help give my website a nice, clean look.

    Learning and Applying

    I knew that the best way to learn coding is by doing, and that's exactly what I did. Everyday, for months, I worked a little bit more on my website all the while studying the fundamentals. I called this method 'Learning and Applying', which meant that no matter how much theory I was learning across Computer Science, I would always be applying what I learnt.

    I failed many times to get things to work, and I especially struggled to get a design that fit how I wanted the website to look. But eventually I got a good rythm going, and the more I learnt, the better I got at coding. The website you see today is a culmination of months of hard work and study, and I'm very proud of the result. There is still much more to be learnt, such as making the website Dynamic using JavaScript. This will take time, but this is what I love doing.
